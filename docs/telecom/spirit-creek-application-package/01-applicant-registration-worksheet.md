## Current primary-source requirements noted

CRTC guidance states telecommunications providers must register before offering service. Local VoIP providers must address 9-1-1 arrangements and registration requirements. A BITS licence is required when carrying telecommunications traffic internationally; the current application process includes Form 503 through the Data Collection System and a signed/notarized affidavit.

These notes are preparation aids, not legal conclusions. Current instructions and forms must be rechecked before filing.
