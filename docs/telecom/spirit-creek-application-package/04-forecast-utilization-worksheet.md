## Initial resource concept

A single 1,000-number block may be evaluated only if the selected rate centre, carrier-hosting arrangement, applicable CNA/CNAC rules, utilization justification, and required identifiers support it. This is not a request or assignment.

## Data required before certification

- Customer acquisition model and service-area demand
- Existing assigned inventory, if any
- Reserved, aging, administrative, and unavailable quantities
- Monthly activation and churn assumptions
- Port-in versus new-number demand
- Months-to-exhaust calculation
- Rate-centre-specific utilization
- Authorized certification and supporting records
