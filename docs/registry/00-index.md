# WW.CX Geographic Registry

## Purpose

Establish a canonical geographic reference layer for WW.CX, Edge1, BigBird, telephony, messaging, scheduling, and operational systems.

## Standards

The registry will be sourced from recognized standards:

- ISO 3166 country identifiers
- ITU-T E.164 telephone calling codes
- IANA Time Zone Database
- ISO 4217 currencies
- Unicode CLDR regional metadata

## Planned Components

- Country registry
- Calling code registry
- Time zone registry
- Currency registry
- Validation tooling
- API exposure for internal services

## Design Goals

- One authoritative source
- Versioned updates
- Machine-readable formats
- Safe integration across services
