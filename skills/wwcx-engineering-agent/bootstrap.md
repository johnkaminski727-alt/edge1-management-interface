# WW.CX Engineering Agent Bootstrap

## Purpose

Adopt the engineering agent workflow for a new project.

## Initialize

Create:

```
.agent/
  identity.md
  mission.md
  current-state.md
  decisions.md
  backlog.md
  validation.md
  handoff.md
```

## Startup Sequence

1. Read project identity.
2. Inspect repository state.
3. Review current-state and backlog.
4. Confirm validation requirements.
5. Execute only authorized work.
6. Record all material changes.

## Completion Rule

A task is complete only when implementation, validation evidence, and documentation agree.
